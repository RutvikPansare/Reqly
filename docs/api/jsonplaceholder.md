# 🌍 JSONPlaceholder API Reference

> JSONPlaceholder starter collection - free fake REST API, no auth, no setup

This document provides a reference for the `jsonplaceholder` collection, covering endpoints for managing todos and users.

---

## 📝 Todos

### `POST` Create a Todo

Creates a new todo item.

**Endpoint:** `{{baseUrl}}/todos`

#### Headers
| Key | Value | Description |
|---|---|---|
| `Content-Type` | `application/json` | Indicates JSON payload |

#### Request Body
```json
{
  "title": "Learn Reqly",
  "completed": false,
  "userId": 1
}
```

---

### `GET` Retrieve a Todo

Fetches a specific todo item by its ID.

**Endpoint:** `{{baseUrl}}/todos/1`

---

### `GET` List Todos

Retrieves a list of all todos, optionally filtered by `userId`.

**Endpoint:** `{{baseUrl}}/todos`

#### Query Parameters
| Key | Value | Description |
|---|---|---|
| `userId` | `{{userId}}` | Filter todos by a specific user's ID |

---

## 👤 Users

### `GET` Retrieve a User

Fetches a user profile dynamically based on the ID created in the previous request.

**Endpoint:** `{{baseUrl}}/users/{{create-todo.response.body.userId}}`
